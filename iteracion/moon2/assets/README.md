# Assets para 911 ES: Turno Lofi

Deja los archivos con estos nombres para que el HTML los cargue automaticamente.

## Imagenes

Carpeta: `assets/images/vignettes/`

- `mareo.png`
- `fuego-cocina.png`
- `asalto.png`
- `inconsciente.png`
- `ruido-patio.png`
- `choque-ruta.png`
- `dolor-pecho.png`
- `ascensor-atrapado.png`
- `olor-gas.png`
- `tienda-robo.png`
- `crisis-panico.png`
- `disturbio-vecinal.png`
- `persona-frio.png`
- `camion-ruta.png`

Formato recomendado: PNG o JPG, 16:9, al menos 1280x720. Sin texto dentro de la imagen.

Si una imagen falta, el juego usa la vineta CSS de respaldo.

## Musica

Carpeta: `assets/audio/music/`

- `night-shift-lofi.mp3`

Recomendado: loop lofi suave, 60-85 BPM, sin voces prominentes, volumen master bajo.

## SFX

Carpeta: `assets/audio/sfx/`

- `call-in.wav`
- `dispatch.wav`
- `unit-return.wav`
- `error-soft.wav`
- `keyboard_sound_writing.mp3`

Recomendado: sonidos cortos, suaves, sin picos agresivos. WAV u OGG funcionan bien; si cambias la extension, actualiza `GAME_SCRIPT.assets` en el HTML.
